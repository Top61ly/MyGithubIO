# H5 Hello World 工程

这是一个简单的HTML5 Hello World演示项目。

## 项目结构

```
H5Test/
├── index.html      # 主HTML文件
├── style.css       # 样式文件
├── script.js       # JavaScript交互脚本
└── README.md       # 项目说明
```

## 功能特性

- 响应式设计
- 漂亮的渐变背景
- 交互式按钮
- 点击计数功能

## 使用方法

1. 在浏览器中打开 `index.html` 文件
2. 点击"点击我"按钮查看交互效果
3. 每次点击都会显示点击次数

## 浏览器兼容性

支持所有现代浏览器（Chrome、Firefox、Safari、Edge等）

## 快速启动

如果你有 Python 安装，可以通过以下命令在本地运行：

```bash
# Python 3.x
python -m http.server 8000

# 然后在浏览器访问 http://localhost:8000
```

或使用 Node.js 的 http-server：

```bash
npx http-server
```
